# Мини CRM – Сопроводительный файл

## 1. Описание проекта
Мини CRM для сбора и обработки заявок с сайта через виджет.

## 2. Сущности
- **User** – менеджер/админ
- **Customer** – клиент
- **Ticket** – заявка (с файлами через Spatie MediaLibrary)

## 3. Основной функционал
- Виджет обратной связи (`/widget`) через Blade + AJAX
- REST API: создание заявки, статистика заявок
- Админка: просмотр, фильтрация, смена статуса заявки
- Ограничение: не более одной заявки на сутки с одного email/телефона

## 4. Используемые технологии и библиотеки
- Laravel 12, PHP 8.4
- Spatie MediaLibrary (файлы)
- Spatie Permission (роли)
- Swagger (API документация)
- Postman для тестирования
- PSR-12, SOLID, KISS, DRY

## 5. Особенности реализации
- Вся логика через сервисы и репозитории
- Валидация через FormRequest
- Enum для статусов заявки
- Eloquent scopes для статистики
